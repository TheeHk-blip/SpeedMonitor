# SpeedMonitor
A lightweight, real-time network speed utility for windows.

# How to Run:
Download the executable from the Assests section.

*Note: 
The app requires the .NET Desktop Runtime to run on your system.

You can right click on the interface to show a context menu of extra functions.

Because this app is an unsigned indie utility, Windows SmartScreen might pop up a blue warning. To run it, click "More info" and then select "Run anyway".
